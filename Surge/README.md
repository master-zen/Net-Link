# Surge Assets

This directory contains the repository's Surge-facing deliverables.

## Contents

- `Module/`: generated Surge modules.
- `Rules/`: generated Surge rule lists.
- `Icon/`: icon source assets and metadata inputs.
- `Scripts/`: generated script assets referenced by published modules.

The files here are primarily outputs. Behavioral changes should usually be implemented in `data/` or `scripts/`, then regenerated.
