# Surge Modules

This folder contains generated Surge module files intended for direct subscription or import.

## Current Focus

- `Ad_Block.sgmodule` is produced by the ad-block automation pipeline.

## Expectations

- Module headers must follow the required Surge-compatible metadata format.
- Rule-based reject directives are extracted into rule lists and should not remain inside the final merged ad-block module.
- Only validated outputs should be published here.
